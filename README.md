# go-pong (in progress)

A game of pong that can be played by two players.

<img src="https://user-images.githubusercontent.com/11229485/160986286-40960b71-c796-4e5d-b424-1039799b4611.JPG" width="400" />

### Controls

##### Player 1

-   Move up: W
-   Move down: S

##### Player 2

-   Move up: Arrow Up
-   Move down: Arrow Down

##### Other

-   Pause: p
-   Exit: q
