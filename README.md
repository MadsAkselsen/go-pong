# go-pong (in progress)

A game of pong that can be played by two players.

### Controls

##### Player 1

-   Move up: W
-   Move down: S

##### Player 2

-   Move up: Arrow Up
-   Move down: Arrow Down

##### Other

-   Pause: p
-   Exit: q
